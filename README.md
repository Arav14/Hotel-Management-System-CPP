# Hotel Management System (C++)

A console-based Hotel Management System developed using C++.

## Features
- Room booking management
- Food ordering system (Pasta, Burger, Noodles, Shakes, Chicken Rolls)
- Inventory tracking
- Sales and revenue tracking
- Daily collection summary
- Menu-driven interface

## Technologies Used
- C++
- OOP Concepts (basic structure)
- Conditional statements
- Switch-case
- Loop control using goto
- Standard Input/Output

## Pricing Structure
- Room: ₹1200
- Pasta: ₹250
- Burger: ₹120
- Noodles: ₹150
- Shake: ₹100
- Chicken Roll: ₹150

## How to Run

1. Install a C++ compiler (MinGW / g++)
2. Compile the program:

   g++ main.cpp -o hotel

3. Run:

   ./hotel
